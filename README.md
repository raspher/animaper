# animaper
Anime/manga scraper, health checker

## Protect targets
- [ ] internationalization
- [ ] scrape data (anime, manga):
  - [ ] item (manga, anime)
    - [ ] cover photo
    - [ ] description
    - [ ] episodes
    - [ ] related series
    - [ ] ratings (just an average and vote numbers)
    - [ ] release date
  - [ ] episodes:
    - [ ] title
    - [ ] media link (optional)
      - [ ] language
      - [ ] quality
- [ ] episodes health check
  - [ ] last date checked
  - [ ] report dead links

### supported platforms
- [ ] shinden
- [ ] myanimelist
- [ ] more in future?
